## ğŸ¯ Topics Covered

### 1. Arrays
- **Two Sum** - Find two numbers that add up to a target
  - Uses HashMap for O(n) solution
  - LeetCode Problem #1

- **Maximum Subarray** - Find the contiguous subarray with the largest sum
  - Implements Kadane's Algorithm
  - O(n) time complexity
  - LeetCode Problem #53

### 2. Linked Lists
- **Reverse Linked List** - Reverse a singly linked list
  - Iterative and recursive solutions provided
  - O(n) time complexity, O(1) space (iterative)
  - LeetCode Problem #206

### 3. Dynamic Programming
- **Fibonacci Series** - Calculate the nth Fibonacci number
  - Multiple approaches: Recursion, Memoization, Tabulation, Space-Optimized
  - Compare time and space complexities
  - LeetCode Problem #509
